# cognition-poc
Cognition Architecture Proof of Concept

### Setup
```commandline
git clone https://github.com/geospatial-jeff/cognition-poc.git
cd cognition-poc
conda env create -f environment.yml
conda activate cognition-poc
```

### Demos & Docs
```commandline
cd demos
jupyter notebook
```
